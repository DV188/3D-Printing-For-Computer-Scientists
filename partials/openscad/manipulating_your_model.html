Manipulating Your Model

Now that we know how to make basic shapes using OpenSCAD's built in functions, let's take a look at ways of manipulating these shapes.

Linear Extrusion

Linear extrusions are the simplest extrusion being discussed. By extruding a two dimensional shape we are essentially pulling the shape into the third dimension to create a new model. We will begin by drawing a two dimensional square to then pull into three dimension in the form of a cube.

linear_extrude.scad
 
/*
 * The linear_extrude function stretches an object based on the parameters provided.
 * Parameters:
 *      height - specifies the length at which the object will be extruded
 *      center - boolean specifying if the extrusion will be centered around the object or not
 *      convexity - specifies the maximum number of front sides (back sides) a ray intersecting the object might penetrate
 *      twist - the degree for which the extrusion is twisted around its height axis
 */
linear_extrude(size = 10, center = true, convexity = 10, twist = 0) {
    square(size = 10, center = true);
}

Substituting a value for the twist parameter and smoothing the object provide results like this:

linear_extrude(size = 10, center = true, convexity = 10, twist = 50, $fn = 200)

Adding the scale parameter will scale the extrusion with respect to the scale size at the end of the extrusion height. The result is a gradually increasing object as one travels towards the end of the extrusion. Scale can also be used asymmetrically using a vector instead of a number.

linear_extrude(size = 10, center = true, convexity = true, scale = 2)

Rotate Extrusion

Rotational extrusions wrap the object around the Z axis following the right hand rule (just like cross product). Rotational extrusions are easily explained with an example:

/*
 * The rotate_extrude function wraps an object around the z-axis.
 * Parameters:
 *      convexity - specifies the maximum number of front sides (back sides) a ray intersecting the object might penetrate
 */
rotate_extrude(convexity = 10) {
    translate([10, 0, 0]) // Move object up 10 along the x-axis.
        square(size = 10, center = true);
}
