Debugging

Debugging in OpenSCAD can be challenging. Compile-time errors are not readily explained and as with any three dimensional object creation sometimes problems are not obvious. OpenSCAD has several modifiers that can be added to your code that can help with debugging.

Background Modifier

The background modifier (%) draws the object onto your model without interacting with the model. Objects affected by the background modifier are drawn in grey.

background_modifier.scad
 
/*
 * The background modifier (%) shows the object as it would be had you proceeded with your actions.
 */

cube([10, 10, 10]);

%translate([5, 5, 5]) // Cube drawn in grey and not actually rendered.
    cube([10, 10, 10]);

The translucent grey cube has been suppressed by the background modifier in our example below. This image is only available in the preview mode and will not show up if the model is fully rendered.

Debug Modifier

When cutting holes or subtracting material from your model, the debug modifier (#) can come in use when you would like to see the part you are modifying. The debug modifier outlines the part in translucent pink to show the object in the preview window.

debug_modifier.scad
 
/*
 * The debug modifier outlines the modified object for easier inspection.
 */

difference() {
    cube([10, 10, 10]);

    #translate([5, 5, 5])
        cube([10, 10, 10]);
}

This time, with the debug modifier, the cube is now translucent pink. The difference taken out of the first cube is still performed but the both parts are displayed for a better view of what OpenSCAD is trying to accomplish.

Root Modifier

The root modifier ( ! ) will only display what is being modified and hide the rest from view.

root_modifier.scad
 
/*
 * The root modifier (!) hides the rest of the model from view.
 */

cube([10, 10, 10]);

!translate([5, ,5, 5])
    cube([10, 10, 10]);

The output of the previous code is the same as the other cubes created. This output, however, does not display the other cube as all output is suppressed other than the object being modified by the root modifier.

Disable Modifier

The final modifier is the disable modifier (*). The disable modifier disables the object being modified, it acts in the opposite way as the root modifier.

disable_modifier.scad
 
/*
 * The disable modifier (*) hides the object from the view.
 */

cube([10, 10, 10]);

*translate([5, 5, 5])
    cube([10, 10, 10]);
