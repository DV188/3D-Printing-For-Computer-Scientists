The Basics

OpenSCAD is a programming language with a familiar C like syntax. Unlike C, however, OpenSCAD is based on a functional programming paradigm that benefits the creation of our models.

Hello World

For our first program, let's begin with a Hello World example. Feel free to use your operating system's file browser or OpenSCAD's new file button. We will be using the terminal (Bash) to manage our files. Let us begin by creating OpenSCAD source code:

$ touch hello.scad

The .scad file extension is where the OpenSCAD source code is to be written. Now, let us add some code to the file we just created.

hello.scad
 
// Comments are the same as C/C++.
/*
 * Multiline comments are also supported.
 */

echo("Hello World");

If you were able to set up our openscad alias mentioned in Programmers Option, you should be able to write the following in your terminal to see the results of the program.

$ openscad hello.scad

Note, the program above does not draw anything to the screen but instead demonstrates how to write to standard output. You should see a line of text echoing "Hello World" in the console window of OpenSCAD. Such a feature can be crucial while trying to debug a pesky bug in your models.

Variables

OpenSCAD is a loosely typed language, variables are not assigned explicit types (think JavaScript). Variable instantiation and assignment is very straightforward as shown below. Arrays are handled using vectors, OpenSCAD offers multiple ways of defining vectors. Vectors show many similarities to lists commonly found in functional languages.

variables.scad
 
my_number = 58; // Assigns an integer value to the variable with provided name.
echo("The value of my_number is: ", my_number);

my_string = "Hello World"; // Assigns a string value to the variable with provided name.
echo("The value of my_string is: ", my_string);

circumference = 2*PI*5.2; // PI is built into the language.
echo("The value of circumference is: ", circumference);

boolean = false; // Boolean values, true or false.
echo("The value of boolean is: ", boolean);

vector = [1, 2, 3, 4, 5]; // Elements of an vector (array) listed verbosely.
echo("The value of vector at position 0 is: ", vector[0], " and position 3 is: ", vector[3]);

Since OpenSCAD follows a functional programming paradigm, variables act similar to constant variables in languages like C. Variables are set at compile-time and not at run-time, they should be considered immutable.
