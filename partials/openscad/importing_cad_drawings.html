Using CAD Drawings

The .dxf file type was developed by Autodesk as a format for saving computer-aided design (CAD) drawings. This format is open source and very popular in a lot of different CAD software packages. If you come across these file types and you would like to 3D print them, OpenSCAD provides functionality to do just that.

Import

Using what we learned to manipulate two dimensional images into three dimensional models, the import function provided by OpenSCAD can take a .dxf file and turn it into much more. The .dxf being loaded is from the Printrbot Starter Kit.
import.scad
Base_Plate_V2_.dxf
 
/*
 * The import function is used to import .dxf file typically created with CAD software.
 */
linear_extrude(height = 10, center = true)
    import (file = "Base_Plate_V2_.dxf");

Our two dimensional CAD drawing is now three dimensional!
