Slicer

We now know how to create a three dimensional model using OpenSCAD. OpenSCAD uses the popular .stl file format for exporting its three dimensional models. It is not typical for consumer 3D printers to take the .stl file format as input for printing the model. STL's contain no information about how to build the model and the intermediate step for doing so is usually done with a separate piece of software.

The software package that takes an .stl file and converts it into 3D printable instructions is known as a slicer. Slicing software, like CAD software, comes in various different forms with both open and closed source options. Some of the most popular slicing software includes, Slic3r, Cura, and Simplify3D. The first two being freely available and open source, Simplify3D being a paid solution. It is recommended that you experiment with different slicers as they can affect your prints as much as the printer itself. Moving forward, we will be discussing Slic3r as it recommended by many different 3D printer manufactures. The Slic3r engine is popular in several slicing software packages, maybe most popular being Repetier-Host.

Repetier-Host And Slic3r

Repetier-Host is a fully featured 3D slicing software package. This piece of software allows the user to take their three dimensional .stl and position it on the 3D printer build platform, slice the model into G-code, preview your model, and print to your 3D printing hardware. Repetier-Host takes advantage of the Slic3r engine (or others) to perform these actions.

When you open Repetier-Host and open it on your machine for the first time, you should see a screen similar to the image above. Towards the left side of the application is the viewing window. Unlike OpenSCAD, The viewing window does not show a traditional Cartesian coordinate but instead it's meant to represent the build area of the printer you have selected in your settings. The right hand side of the screen has some options for positioning, rotating, and scaling, your object. This allows for further manipulation of you CAD three dimensional model with respect to your printer. It also allows for multiple models to be printed and positioned simultaneously. Towards the bottom, a console output window is available which will become especially useful for troubleshooting physical printing problems.

Printer Setup

Before we can calibrate the slicer, we must inform Repetier-Host of our printer properties before we can make any 3D prints. Navigating to Printer > Printer Settings, a pop up window with printer settings will appear.

RepRap printers will typically have documentation that details these settings for the printer you may be using. It is important to use the recommended settings as a base point and dial in the setting from there. The printer settings range from extruder temperatures to bed size and feed rate. All the parameters should be filled out before further calibration is completed.

Slicer Calibration

The slicer is responsible for most of the software calibration required to 3D print your models created by CAD software. The slicer's role in print quality is as important as the physical printer itself most of the time. Repetier-Host comes prepackaged with the Slic3r three dimensional slicing engine. Selecting the Slicer tab in the rightmost menu, you can see some settings for different slicers, print settings, and extruder settings.

Selecting the configure button will open up the Slic3r interface to adjust you slicing settings. A large number of options and settings are available through the Slic3r interface. Slic3r does a very good job at detailing what each of these settings do. Simply hover your mouse over various options to see a quick description of that option.
